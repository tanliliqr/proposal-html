# 求婚网页
这是一个html+js+css的求婚网页 
## 分为 恋爱时间轴 恋爱故事 照片长廊 视频助力 爱的誓言 新纪元 等模块
### 恋爱时间轴
内容设计为第一次与女友做的事情 
- 比如初见
- 第一次旅行
- 第一次看雪、海、日出等
### 恋爱故事
可以从多个角度来展开 作者与女友一起九年  每年分为了一个段落
### 照片长廊
这个模块设计的为 所有的照片点滴

### 视频助力
作者收集了双方的长辈、一起的朋友的祝福视频

### 爱的誓言
你的誓言

### 新纪元
新角色 男友成为未婚夫、女友变为未婚妻 

## 拥有的效果
- 背景音乐 可选与自动播放
- 同意时切换高潮音乐
- 爱心、文字等飘落
- 无法选择的拒绝按钮
- 图片自动下拉
- 视频自动切换与跳过
- 文案的缓慢输出
